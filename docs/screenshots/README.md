# Screenshots

Drop dashboard screenshots into this folder so the README and other docs have something to show.

Suggested filenames (referenced from the docs):

- `dashboard-preview.png` — the main dashboard view, used in the top of `README.md`
- `concepts-kanban.png` — the concepts-tier kanban
- `projects-kanban.png` — the project-status kanban
- `day-board.png` — the day board with a couple of slots populated
- `timeline.png` — the timeline view
- `stats.png` — the stats panel

Recommended size: 1600px wide minimum. PNG with a light background works well in both light and dark GitHub themes.

How to capture (quick path):

1. `./dashboard/ev-dash` to launch
2. Open `http://localhost:8077` in a browser
3. Use your OS's screenshot tool to grab the relevant view
4. Save it here with the matching filename

The repo ships without any actual images so the project stays small for clones. Adding your own is purely cosmetic — nothing in the code references these files.
