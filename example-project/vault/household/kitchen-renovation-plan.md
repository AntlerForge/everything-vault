# Phased plan

The plan I'm running. Lives separately from the project article so the
project page can stay lean and just point at this.

## Sequence and reasoning

1. **Demo.** Strip back to brick and joists. Confirm what you're working
   with before ordering anything. Skip-hire booked for the start.
2. **Plumbing first-fix.** Reroute supply and waste before any plastering
   goes on the walls. Capped off and pressure-tested.
3. **Electrical first-fix.** Sockets, lighting circuits, induction-hob
   feed. Do this only after the cabinet layout is locked, because socket
   positions depend on it.
4. **Plastering.** A clean two-day stretch — leave the walls alone for at
   least 5 days afterward to dry properly.
5. **Tiling.** Floors first, then any wall tiles. Use spacers; don't rush
   the levelling.
6. **Cabinet fitting.** Carcasses first, doors last. Check level on every
   carcass; old houses lie about being level.
7. **Worktops, splashback, finishing.** Worktop template only after
   cabinets are in. Splashback fits to worktop, not to wall.

## Decisions to lock before each phase

| Phase           | Decisions due                                 |
|-----------------|-----------------------------------------------|
| Plumbing        | Sink position, dishwasher position            |
| Electrical      | Socket count and positions, hob type          |
| Tiling          | Tile choice, grout colour, layout pattern     |
| Cabinets        | Cabinet brand, finishes, handle choice        |
| Worktops        | Material, edge profile, splashback height     |

## What I learned the hard way (last bathroom)

- Order tiles before plastering, not after. Lead times are real.
- Don't let the plasterer and the tiler overlap. Plaster needs to dry.
- The contingency line will be spent. Plan for it.
