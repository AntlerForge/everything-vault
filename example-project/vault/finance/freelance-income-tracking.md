# Freelance income tracking

A single spreadsheet, one tab per financial year, that tracks invoices,
expenses, VAT, and projected tax. Updated weekly on a Sunday.

## Tabs

1. **Invoices** — one row per invoice. Columns: invoice no., client, date
   issued, date paid, net, VAT, total, status.
2. **Expenses** — one row per receipt. Columns: date, category, vendor,
   net, VAT, total, project tag.
3. **Mileage** — date, from, to, miles, project tag.
4. **VAT summary** — rolling totals per quarter.
5. **Tax projection** — running estimate of payment-on-account due.
6. **Year-end** — totals and journal entries copied to the accountant.

## Weekly workflow

Sunday evening, 20-30 minutes:

1. Pull receipts from the inbox folder.
2. Enter into Expenses tab. Tag against the project that incurred them.
3. Update Invoices tab — anything paid this week.
4. Reconcile against the business bank account.
5. Update the VAT summary.

## Quarterly workflow

Add a 60-minute slot to the calendar at the end of each quarter:

- VAT return prep.
- Tax projection update.
- Quick review of where time vs revenue is going by client.

## Year-end

Hand the year-end tab to the accountant by mid-April. Receipts archive
exported to PDF and filed.

## Notes

- Real account numbers, login details, and HMRC references live in the
  password manager, not here.
- Receipts pile up quickly — letting it slip a month turns a 30-minute job
  into a 4-hour one. Don't skip the Sunday slot.
